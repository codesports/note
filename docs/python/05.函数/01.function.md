# 函数

## 概念

在 Python 中，函数是组织好的、可重复使用的代码块，用于实现特定的功能。使用函数可以提高代码的复用性和可维护性，避免重复编写相同的代码。

## 作用

使用函数可以加强代码的复用性，提高程序编写的效率。

1. 提高代码复用性
2. 增强代码可维护性
3. 实现模块化编程
4. 参数化编程
5. 返回计算结果
6. 支持代码抽象

## 语法

在 Python 中，使用 `def` 关键字来定义函数，语法格式如下：

```python
def 函数名（参数列表）:
    函数体
```
